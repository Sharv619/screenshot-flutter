# Product Requirements Document (PRD) - Screenshot Auto-Categorizer
**Version:** 1.1 (Updated: 20260504125917)

## 1. Introduction
The Screenshot Auto-Categorizer is a mobile application built with Flutter that automatically organizes screenshots. It uses on-device machine learning (OCR and Image Classification) to extract actionable information from screenshots and categorize them into meaningful folders.

## 2. Updated Core Features (v1.1)
### 2.1. Intelligent Action Pop-ups
- **Location Extraction:** Identify addresses/restaurants and provide a Google Maps preview.
- **Book Extraction:** Identify titles/authors and provide links to Amazon and Audible.
- **Action Buttons:** "Save to List", "Open with GMaps", "View on Amazon", etc.

### 2.2. Post-Extraction Deletion (The "Cleanup" Prompt)
- **Feature:** Suggest deleting the original screenshot from the device gallery once information is successfully extracted and saved.
- **Workflow:** Triggers after a user performs an "Action" (Save/Open Link).
- **Security:** Uses native OS-level prompts (PhotoKit on iOS, MediaStore on Android) for user confirmation.

## 3. Technical Requirements
### 3.1. Permissions
- **iOS:** NSPhotoLibraryUsageDescription (Read/Write access for deletion).
- **Android:** READ_MEDIA_IMAGES (Android 13+), WRITE_EXTERNAL_STORAGE (Android 12-).
### 3.2. APIs & Enrichment
- **Google Books API:** For book metadata enrichment.
- **Google Places API:** For restaurant/location details and previews.

## 4. Financial Structure (Technical)
